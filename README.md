# 🏏 IPL 2026 Data Hub

Welcome to the **Nexus Plays** IPL Analytics engine! This project uses Python to transform match data into a visual dashboard.

## 🚀 Features
* 📊 **Interactive Charts:** Visualizing scores using Matplotlib.
* 🏆 **Live Points Table:** Automatically calculates rankings.
* 🏏 **Match Insights:** Detailed breakdown of all 41+ matches.

## 🛠️ Tech Stack
* **Language:** Python
* **Frontend:** Streamlit
* **Visuals:** Matplotlib
